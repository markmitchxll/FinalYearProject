Sound effect files go here.

Required files:
- discovery.mp3    — played when the user finds a new star/planet/constellation
- pet_happy.mp3    — played when the pet reacts happily to a discovery
- evolution.mp3    — played when the pet evolves to the next stage
- tap.mp3          — short click for button presses

Keep file sizes small (< 100KB each) for fast loading on mobile.
MP3 format is supported on both iOS and Android.
